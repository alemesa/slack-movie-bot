
# slack-movie-bot-jam3

Get it [here](https://slack-movie-bot-jam3.herokuapp.com)


To contribute to this repo or do something similar
* First configure Slack
```bash
git clone <repo>
cd <folder>
npm install
nodemon index or node index
```
* Heroku
```
git commit <changes>
git push
git push heroku master
```
* Check Heroku Logs
```bash
heroku logs --tail
```
